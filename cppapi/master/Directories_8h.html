FO_COMMON_API const boost::filesystem::path GetUserConfigDir ( )

Returns the directory where FreeOrion should store user specific configuration data , like the configuration file. Under Unix, this would be $XDG_CONFIG_HOME/freeorion, under Windows, this might be something along the lines of C:\Documents and Settings\Username\FreeOrion or even \\Gandalf\Users\Frodo\Settings\FreeOrion.

Note
  • If the directory does not exist, it will be created. Under Windows and OSX it is the same as the GetUserDataDir()
  • This directory can be considered writable!

Referenced by GetConfigPath(), GetPersistentConfigPath(), and anonymous_namespace{AIWrapper.cpp}::GetUserConfigDirWrapper().
