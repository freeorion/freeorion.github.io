Detailed Description

Encapsulates a variable-length char buffer containing a message to be passed among the server and one or more clients.

Note that std::string is often thread unsafe on many platforms, so a dynamically allocated char array is used instead. (It was feared that using another STL container of char might misbehave as well.)
