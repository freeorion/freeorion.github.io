Detailed Description

ScanlineRenderer renders scanlines in circular/square/arbitrary areas. It loads the scanline shader on first use. There is only expected to by one ScanlineRenderer per compilation unit so that the shader is compiled once.

Bracketing OpenGL primitives with StartUsing() and StopUsing() will render scanlines in that arbitrary area.
