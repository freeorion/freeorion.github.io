Detailed Description

Represents a CUIToggleButton.

Renders a SubTexture depending on the checked state of the CUIToggleButton. When the button has single texture and is unchecked, it renders the same icon recolored towards grey. If a second SubTexture is assigned to the button, it will render the appropriate SubTexture. When a mouse pointer hovers over this button, the opposing SubTexture and color are used and highlighted
