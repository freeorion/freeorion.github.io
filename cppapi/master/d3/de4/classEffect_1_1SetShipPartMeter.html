Detailed Description

Sets the indicated meter on all ship parts in the indicated subset. This has no effect on non-Ship targets. If slot_type is specified, only parts that can mount in the indicated slot type (internal or external) are affected (this is not the same at the slot type in which the part is actually located, as a part might be mountable in both types, and located in a different type than specified, and would be matched).
