◆ GlobalSerializationEncodingForEmpire()

int& GlobalSerializationEncodingForEmpire ( )

This must be set to the encoding empire's id when serializing various gamestate information, so that only the relevant info is serialized for the intended recieipient.

This is implemented this way so that we don't need to write custom boost::serialization classes that implement empire-dependent visibility.
