Detailed Description

Returns obj iff obj is a Fleet belonging to the given empire that is parked at a System, not under orders to move.

If the given empire is ALL_EMPIRES, all orderd moving fleets will be returned. Note that it is preferable to use this functor on System searches, rather than Universe ones.
