◆ RootCandidateInvariant()

virtual bool Condition::Condition::RootCandidateInvariant ( ) const
inlinevirtual

Returns true iff this condition's evaluation does not reference the RootCandidate objects.

This requirement ensures that if this condition is a subcondition to another Condition or a ValueRef, this condition may be evaluated once and its result used to match all local candidates to that condition.
