void PythonBase::HandleErrorAlreadySet ( )

Handles boost::python::error_already_set. If the error is SystemExit the python interpreter is finalized and no longer available.

Call PyErr_Print() if the exception is an error.

HandleErrorAlreadySet is idempotent, calling it mutliple times won't crash or hang the process.

References ErrorLogger, Finalize(), and m_system_exit.
