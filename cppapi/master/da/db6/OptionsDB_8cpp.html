bool RegisterOptions ( OptionsDBFn  function)

adds function to a vector of pointers to functions that add Options to the OptionsDB. This function returns a boolean so that it can be used to declare a dummy static variable that causes function to be registered as a side effect (e.g. at file scope: "bool unused_bool = RegisterOption(&foo)").
