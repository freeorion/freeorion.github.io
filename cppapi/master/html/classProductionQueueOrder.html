Detailed Description

The Order subclass that represents changing an empire's production queue. The ProductionItem ctor adds the build to the beginning or end of empire's queue, the 3-arg ctor moves an existing build from its current location at index to a new one at new_index, and the 2-arg ctor removes the build at index from empire's queue.
