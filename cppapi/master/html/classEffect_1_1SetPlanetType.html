Detailed Description

Sets the planet type of the target to type.

This has no effect on non-Planet targets. Note that changing the type of a PlanetType::PT_ASTEROID or PlanetType::PT_GASGIANT planet will also change its size to PlanetSize::SZ_TINY or PlanetSize::SZ_HUGE, respectively. Similarly, changing type to PlanetType::PT_ASTEROID or PlanetType::PT_GASGIANT will also cause the size to change to PlanetSize::SZ_ASTEROID or PlanetSize::SZ_GASGIANT, respectively.
