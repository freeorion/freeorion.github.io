Detailed Description

Sets the planet size of the target to size.

This has no effect on non- Planet targets. Note that changing the size of a PlanetType::PT_ASTEROID or PlanetType::PT_GASGIANT planet will also change its type to PlanetType::PT_BARREN. Similarly, changing size to PlanetSize::SZ_ASTEROID or PlanetSize::SZ_GASGIANT will also cause the type to change to PlanetType::PT_ASTEROID or PlanetType::PT_GASGIANT, respectively.
