Gets called for each link of the type this decorator is assigned to.

The return value is shown to the user as the link. The default implementation wraps content in an rgba tag that colors it by ClientUI::DefaultLinkColor

Parameters
targetThe target of the link. Usually an id of something or the name of an encyclopedia entry.
contentThe text the link tag was wrapped around.
