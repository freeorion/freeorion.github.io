Detailed Description

JumpDistanceSys1Visitor and JumpDistanceSys2Visitor are variant visitors that can be used to determine the distance between a pair of objects locations represented as GeneralizedLocations. JumpDistanceSys2Visitor determines the distance between _sys_id1 and the GeneralizedLocation that it is visiting.
