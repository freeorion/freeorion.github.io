Detailed Description

template<typename T, typename V = T>
struct ValueRef::Statistic< T, V >

The variable statistic class.

The value returned by this node is computed from the general gamestate; the value of the indicated property_name is computed for each object that matches sampling_condition and the statistic indicated by stat_type is calculated from them and returned.
