Detailed Description

Matches a specified number of objects that match Condition condition or as many objects as match the condition if the number of objects is less than the number requested.

If more objects match the condition than are requested, the objects are sorted according to the value of the specified property_name and objects are matched according to whether they have the specified sorting_type of those property values. For example, objects with the largest, smallest or most common property value may be selected preferentially.
