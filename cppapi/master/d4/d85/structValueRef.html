Detailed Description

template<class T>
struct ValueRef< T >

the constant value leaf ValueRef class.

The variable value ValueRef class. The value returned by this node is taken from the gamestate, most often from the Source or Target objects.

The variable statistic class. The value returned by this node is computed from the general gamestate; the value of the indicated property_name is computed for each object that matches sampling_condition and the statistic indicated by stat_type is calculated from them and returned.

The complex variable ValueRef class. The value returned by this node is taken from the gamestate.

The variable static_cast class. The value returned by this node is taken from the ctor value_ref parameter's FromType value, static_cast to ToType.

The variable lexical_cast to string class. The value returned by this node is taken from the ctor value_ref parameter's FromType value, lexical_cast to std::string

Looks up a string ValueRef and returns the UserString equivalent.

Returns the in-game name of the object / empire / etc. with a specified id.

An arithmetic operation node ValueRef class. One of addition, subtraction, mutiplication, division, or unary negation is performed on the child(ren) of this node, and the result is returned.
