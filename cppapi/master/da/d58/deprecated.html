Member StringFromFlags (unsigned int flags)
All the GG XML classes are deprecated and will be removed upon the next major release. Takes an unsigned integer that is the result of the bitwise OR-ing of enumerated values of type T, and generates a string of the form "value1 value2 value3 ...". This function will yield the zero-value of enum type T (e.g. "TF_NONE") if no nonzero flags are specified. If no such zero-value exists, the returned string will be empty.
Note
This function depends on EnumMap<T> being defined.
