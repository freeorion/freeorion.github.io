void ColonizeOrder::ExecuteImpl ( ) const

Preconditions:

  • m_planet must be the ID of an un-owned planet.
  • m_ship must be the the ID of a ship owned by the issuing empire
  • m_ship must be the ID of a ship that can colonize and that is in the same system as the planet.

Postconditions:

  • The ship with ID m_ship will be marked to colonize the planet with id m_planet during the next turn processing.
