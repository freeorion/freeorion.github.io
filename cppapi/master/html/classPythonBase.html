◆ HandleErrorAlreadySet()

void PythonBase::HandleErrorAlreadySet ( )

Handles boost::python::error_already_set.

If the error is SystemExit the python interpreter is finalized and no longer available.

Call PyErr_Print() if the exception is an error.

HandleErrorAlreadySet is idempotent, calling it multiple times won't crash or hang the process.
