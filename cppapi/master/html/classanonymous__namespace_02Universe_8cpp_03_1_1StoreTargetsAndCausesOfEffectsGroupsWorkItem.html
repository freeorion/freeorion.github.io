Detailed Description

Used by GetEffectsAndTargets to process a vector of effects groups.

Stores target set of specified effects_groups and source_object_id in targets_causes NOTE: this method will modify target_objects temporarily, but restore its contents before returning. This is a calleable class instead of an ordinary method so that we can use it as work item in parallel scheduling.
