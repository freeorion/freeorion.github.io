Detailed Description

Functions like a StaticGraphic, except can have multiple textures rendered on top of eachother, rather than just a single texture.

creates a MultiTextureStaticGraphic from multiple pre-existing Textures which are rendered back-to-front in the order they are specified in textures with GraphicStyles specified in the same-indexed value of styles. if styles is not specified or contains fewer entres than textures, entries in textures without associated styles use the style GRAPHIC_CENTER.
