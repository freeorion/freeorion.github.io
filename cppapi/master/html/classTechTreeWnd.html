Detailed Description

Contains the tech graph layout, some controls to control what is visible in the tech layout, the tech navigator, and the tech detail window.

TechTreeWnd contructor is usually called before client has access to techs. Attempting to show the tech tree takes a long time and generates errors. If initially_hidden is true then the tech categories are not parsed until the first time Show() is called, speeding up the constructor and preventing spurious errors.
