Detailed Description

Generates a sitrep message for the empire with id recipient_empire_id.

The message text is the user string specified in message_string with string substitutions into the message text as specified in message_parameters which are substituted as string parameters %1%, %2%, %3%, etc. in the order they are specified. Extra parameters beyond those needed by message_string are ignored, and missing parameters are left as blank text.
