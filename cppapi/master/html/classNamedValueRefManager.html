unsigned int NamedValueRefManager::GetCheckSum ( ) const -> unsigned int

Returns a number, calculated from the contained data, which should be different for different contained data, and must be the same for the same contained data, and must be the same on different platforms and executions of the program and the function.

Useful to verify that the parsed content is consistent without sending it all between clients and server.
