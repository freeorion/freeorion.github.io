Detailed Description

Returns obj iff obj is a Fleet belonging to the given empire, and that is under orders to move, but is not yet moving.

If the given empire is ALL_EMPIRES, all stationary fleets will be returned.

Note that it is preferable to use this functor on System searches, rather than Universe ones.
