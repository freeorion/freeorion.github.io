Detailed Description

specifies a single item of game content that may be unlocked for an empire. The type field stores the type of item that is being unlocked, such as a building or ship component, and the name field contains the name of the actual item (e.g. (UIT_BUILDING, "Superfarm") or (UIT_SHIP_PART, "Death Ray")).
