void ShutdownLoggingSystemFileSink ( )

Shutdown the file sink. This should be called near the end of main() before the start of static de-initialization.

The file sink may not be safe to use during static deinitialization, because of the following bug:

http://www.boost.org/doc/libs/1_64_0/libs/log/doc/html/log/rationale/why_crash_on_term.html https://svn.boost.org/trac/boost/ticket/8642 https://svn.boost.org/trac/boost/ticket/9119

When either ticket is fixed the ShutdownLoggingSystemFileSink() function can be removed.
