Detailed Description

LazyScrollerLinkText is a link text that initially populates itself with an ellipsis.

As a one time effect, when it comes into view it populates itself with the text which will be processed by DetermineLines and flowed by the Layout.

This works around problems with Font::DetermineLinesImpl() which is too costly to run on 100 combat reports without freezing the UI for an extended portion of time.

This assumes:

  • CombatLogWnd is in a TabWnd as the second tab.
  • CombatLogWnd is in a ScrollPanel
